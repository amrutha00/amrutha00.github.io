Smart Pantry

Github Link

React.js, Node.js, Express.js, MongoDB, Firebase, GPT-3.5

  • Developed an intuitive food pantry management system using Firebase, MongoDB, and React Material UI, allowing users to seamlessly add, track, and remove food items with expiry dates while ensuring automatic removal of expired items based on timezone settings.
  • Implemented a marketplace feature enabling users to list surplus items for sale, with contact details provided for seamless transactions, enhancing user engagement and facilitating easy purchasing via a user-friendly cart system displaying prices.
  • Enhanced user experience by integrating a recipe recommendation feature based on stored food items, offering personalized cooking suggestions and promoting efficient utilization of available inventory.
